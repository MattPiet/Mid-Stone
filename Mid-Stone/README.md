# Midstone Project

## How to run

- Set the running mode to `Debug | x86`
- Ensure that you have `GameDevM` downloaded and stored in your C: Drive
- If you have issues with environment variables, edit your run configuration to have the following environment values:
  ```Path=C:\GameDevM\OpenGL\dll\;C:\GameDevM\SDL\dll\;```
